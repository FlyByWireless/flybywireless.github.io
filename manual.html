1. Setup Instruction

Step 1: Download the iOS app

You can download the Fly-By-Wireless app from the apple app store. The app currently does not support android device.

Step 2: Download & Install vJoy Device Driver

Fly-By-Wireless is powered by the opensource vJoy project. vJoy is a device driver that bridges the gap between any device that is not a joystick and an application that requires a joystick. The latest vJoy device driver is available for download from the link below, or you can go to the official site of the vJoy project for more versions.

Download vJoy Device Driver for Windows (x64)

When the installer prompt with the Select Components option, make sure you select the vJoy Configuration application. Other companion applications is not required by Fly-By-Wireless.

Step 3: Download & Install Fly-By-Wireless Server

Fly-By-Wireless server is required to be installed on the PC you use to run flight simulator. The server application receives flight control commands from the Fly-By-Wireless app and feeds them to the vJoy Device Driver. The latest server is available for download from the link below.

Download Fly-By-Wireless Server for Windows (x64)

2. Normal Procedure

Fly-By-Wireless is easy to use. Before each flight, you only need to take less than a minute to get things to work.

First, launch the Fly-By-Wireless server on your computer. Before enabling it, check if the server is configured correctly:

  1. Server IP: The application automatically detects your computer's local IP address. If your computer has only one network interface, you don't need to anything. If your computer has multiple network interface, there can be multiple IPs. In this case, you may want to select IP address manually in the drop down menu. Make sure you select the IP address that correspond to the WLAN your phone connects to.
  2. Server Port: The default port that the server listens on is 7773. If some other application is taking this port (The application will prompt with error in this case), you can manually change the port number to something else between 1000-65535.
  3. vJoy Device: Select the vJoy device you want to use. The default device is #1, leave it unchanged unless you're doing some advanced configuration. If there's no device to select from in the drop down menu, make sure you have vJoy device driver correctly installed and refresh with Main-Refresh vJoy Device List.

Next, check Enable checkbox to start the server. After the server is enabled, start the app, enter the server's IP and port on the main screen, than tap the orange button to start.

Closing the server window will minimize the it to system tray. If you want to exit, right click the tray icon, or using the Main menu option when the window is showing.

3. Control Binding

The vJoy device is seen by Windows as a standard joystick device. Use the Configure vJoy companion app you installed when in Section 1 - Step 2 to configure the number of axes, sliders, buttons, and POVs the vJoy device have. As a minimum requirement, the current version of Fly-By-Wireless uses 3 Axis (X, Y, Z) , a Slider (Slider), 8 Buttons, and a 4-direction POV. When configuring your primary vJoy device, make sure the minimum requirement is met.

Just like using a physical joystick, you also need to bind the joystick's control input to flight simulation events. We provided some examples of control bindings, but you're always free to make your own.

4. App Interface & Tips

  1. Navigation Bar:
    You can access User Manual and Settings from the navigation bar.
  2. Network & Enable Button:
    IP & port box is where you input server address; The big button below is the switch that toggles network functionality.
  3. Control Visualizer:
    The X, Y, Z axis inputs are visualized in this area, tap the yoke in the middle can disable/enable motion sensor (X/Y axis).
  4. Left Buttons:
    Button 1, 3, 5, 6.
  5. Right Buttons & POV:
    Button 2, 4, 7, 8, and 4-way POV.
  6. Sliders & Calibration Button:
    The left slider controls the first slider of the vJoy device; The right slider controls the Z-axis of the vJoy device. The right slider is intended to be used for rudder control, which automatically centers when no input is provided; Tap calibration button in the middle to center X/Y axis with current device attitude.
Tips
  • During a long flight, if you're tired of holding your device, tap the yoke icon in the center of the app to temporary disable the X/Y axis. The current elevator and aileron input will hold until you tap again to re-enable the X/Y axis. The device will automatically calibrate to make sure the input is continuous.
  • You can adjust axis sensitivity in the settings. Generally, use higher sensitivity for X (roll) axis and lower sensitivity for Y (pitch) axis can make adjustment during a landing easier.
